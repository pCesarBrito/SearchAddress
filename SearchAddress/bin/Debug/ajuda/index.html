Apresentação do Programa Search Address!

O programa Search Address é bem simples, mais muito eficiente,útil e intuitivo.Ele foi desenvolvido para auxiliar na busca de endereços em diversas regios do Brasil. O programa"Serach Address"conta com um Banco de Dados interno, sql Server Compact da MicroSoft,uma base de dados compactada, muito completa e robusta, capaz de guardar uma quantidade nuito grande de dados, aproximadamente, mais de dez milhões de Registros.

O banco de dados do programa inicialmente, vem com 10 mil registros cadastrados, mais você tem a possibilidade de manter ou apagar os dados nele contidos, isso vai de acordo com necessidadede de cada úsuario.

O programa conta com um sistema de cadastro automático, cada vez que é colocado um cep para pesquisar um endereço,caso esse endereço não esteja cadastrado na banco de dados, ele é cadastrado automaticamente.Conta também com sitema de vizualização de Ruas do Google maps, onde você pode ver e navegar pela rua procurada. Você também podera ver outras funcionalidades do programa em outros tópicos deste manual.
